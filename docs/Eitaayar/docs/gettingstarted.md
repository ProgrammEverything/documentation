# Installation
To install EitaaAPI You need the PIP package manager: \
To install EitaaAPI using the PIP package manager enter on your command line:
* For windows :
```
py -m pip install EiaaAPI
```
* For Linux - MacOS , ... :
```
python3 -m pip install EitaaAPI
```

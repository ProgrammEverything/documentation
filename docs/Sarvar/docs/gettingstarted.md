## Installation
(REQUIRES PYTHON>=3.10)\
How to install on windows:\
    `py -m pip install Sarvar`\
How to install on mac, linux, ... : (PIP HAS TO BE INSTALLED)\
`python3 -m pip install Sarvar`\
or\
`pip3 install Sarvar`
